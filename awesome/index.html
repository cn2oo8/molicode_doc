# Awesome

一些工具和模板说明。

# 通用列表

# 1.ace前端模板

基于bootstrap aceadmin前端的模板，底层使用mybatis, spring mvc。

    <artifactId>autocode-template-aceadmin</artifactId>
    <groupId>com.jd.b2b</groupId>
    <version>1.0-SNAPSHOT</version>
    <description>基于bootstrap aceadmin前端的模板，底层使用mybatis, spring mvc</description>

# 2.VUE页面模板

用于生成vue列表和增删改查功能，需要一些基础脚手架才能正常运行。 脚手架信息待补充...

    <artifactId>autocode-template-vue</artifactId>
    <groupId>com.jd.b2b</groupId>
    <version>1.0-SNAPSHOT</version>
    <description>VUE页面生成功能</description>

# 3.模板开发测试

工具开发测试使用，可用于演示测试等。

      <groupId>com.jd.b2b</groupId>
      <version>1.0-SNAPSHOT</version>
      <artifactId>autocode-template-devtest</artifactId>
      <name>moliCode模板开发测试</name>
      <description>
          用于版本测试之用；非稳定版本！
      </description>

# 4.moliCode自开发模板

moliCode生成自有代码的模板，主要还是mybatis, spring mvc, Vue界面等。

         <artifactId>autocode-template-selfsdk</artifactId>
         <groupId>com.jd.b2b</groupId>
         <version>1.0-SNAPSHOT</version>
         <description>moliCode生成自有代码的模板</description>
